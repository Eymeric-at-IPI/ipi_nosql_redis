Redischat

Instante tchat app build with redis

Cette application a été conçu pour servir de démonstration technique de la technologie NoSQL Redis.

Celle-ci est développé en Python grâce au Framework Django. Les interactions websocket sont géré via l'outil Channels.
La base de donnée Redis a deux objectif :
-> servir de cache pour Channels
-> servir de stockage pour les messages des différents salons.
